# Firebase Storage
Firebase Storage code guideline for Android developers

## Prerequisites
* Supported Android 4.1 or newer
* Android Studio 3.3.2 or higher
* google-services.json in app-level folder

## Features
* No Authentication usage
* Create references
* Upload files
* Download files
* Use file metadata
* Delete files
