# Firebase Storage
PDF upload and download android app using firebase
## Prerequisites
* Supported Android 4.1 or newer
* Android Studio 3.3.2 or higher
* google-services.json in app-level folder

## Features
* Authentication usage
* Create references
* Upload files
* Download files
